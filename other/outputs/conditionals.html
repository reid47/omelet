Conditionals

Omelet allows for the conditional display of certain blocks of text through its if-elif-else expressions. The predicate in the if and elif cases must be either an interpolation or a parenthetical.

If the predicate is a boolean with value true or a string with value "true", the "then case" will be rendered. Otherwise, the predicate is considered false. Any following elif cases will then be checked. If all are false, the else case will be rendered, if one is present. If there is no else case, nothing will be rendered.

Note that there are several filters which return booleans. In the example below, we use lt less than, which takes a single number as a parameter.

def number = 47
(p ::
    (if {number | lt 10} ::
        The number is less than ten.)
    (elif {number | lt 100} ::
        The number is less than one-hundred but not less than ten.)
    (else ::
        The number is not less than one-hundred.))
<p>The number is less than one-hundred but not less than ten.</p>
